## ðŸ”„ Release Process

```bash
# 1. Sync all files first
./scripts/sync-browsers.ps1   # (if available)

# 2. Update version in ALL manifests
# manifest.json, browsers/chrome/manifest.json, browsers/edge/manifest.json

# 3. Commit changes
git add .
git commit -m "v1.X.X - Description of changes"

# 4. Create and push tag (triggers GitHub Actions)
git tag v1.X.X
git push origin main
git push origin v1.X.X
```

## ðŸš« Common Mistakes

1. **Version mismatch** - Root manifest at 1.3.0 but Chrome at 1.1.0
2. **Forgot to sync** - Changed popup.js in root but not in browser folders
3. **Tag without v** - Using `1.3.0` instead of `v1.3.0` won't trigger release
4. **Pushing ZIP files** - These should be generated by GitHub Actions, not committed
